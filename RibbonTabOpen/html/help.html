Назначение программы

Данный раздел включает описание назначения программы и краткое описание ее функциональных возможностей.

О программе

Программа обогрева нефтяных скважин позволяет проводится теплотехнический расчет работы нефтяной скважины на основе данных технологической анкеты.

Одним из эффективных методов борьбы с отложениями парафина и АСПО в нефтегазовой промышленности является использование электрообогрева.

Программа позволяет расчитать электротехнические параметры нагревательного кабеля для поддерания нужной температуры на устье скважины.

Функциональные возможности

Для расчета вводятся начальные данные по нефтяной скважине и проводится сам расчет.

По результатам расчета строится геотерма скважины. Определяются теплопотери в грунт.

Строится термограмма скважины с учетом характера движения жидкости. Определяется глубина отложения парафинов, а также проводится определение основных характеристик и параметров электрического нагревательного кабеля.

Выходными параметрами программы являются:

- глубина обогрева (длина горячей и холодной зоны);

- электротехнические параметры нагревательного кабеля (мощность обогрева, ток, напряжение и т.д.);

- температуры жилы и оболочки.

Инструкция по эксплуатации

В данном разделе рассмотрена последовательность работы с программой. Рассмотрим на примере как проводить расчет обогрева скважины.

Пример:

1. Вызываем основное окно программы (Рис.1).



В этом окне на панели управления выбираем первую слева иконку. При наведении на нее указателя мышки появится всплывающая подсказка - Расчет обогрева скважины с ЭЦН (см. Рис.2)



Нажимаем на иконку и вызываем функциональное окно программы (см. Рис.3).

2. Функциональное окно программы с новым расчетом открывается на вкладке Инструменты, которая содержит все необходимые инструменты для расчета обогрева скважины

(от ввода исходных данных до обработки результатов и формирования итогового отчета).



Все иконки на ленте, за исключением иконки Ввод данных, будут неактивны.

3. Для ввода исходных данных из технологической анкеты скважины вызываем отдельное окно Ввод данных (Рис. 4).



В этом окне есть несколько возможностей ввода данных:

1. Загрузить ранее сохраненные файлы (кнопка Загрузить данные в нижнем левом углу окна).

При выборе этого способа загрузки данных появится диалоговое окно, в котором необходимо выбрать файл с данными для загрузки в форму.

После этого данные появятся в соответствующих полях и редактор автоматически проверит загруженные данные на валидность (Рис.5).



2. Ввести новые данные в поля анкеты.

В процессе ввода данных в поле, редактор, в автоматическом режиме, проверяет валидность данных.

Если значение будет находится вне допустимого диапазона, то изменится цвет значения (вместо черного будет красный).

Кроме того появится всплывающая подсказка с предупрежением (пример на Рис.6).



После ввода и проверки данных их необходимо сохранить. Для этого в нижнем левом углу формы есть кнопка Сохранить данные (Рис.7).



После нажатия на кнопку Сохранить данные появляется дополнительная форма, после заполнения которой необходимо нажать кнопку Сохранить.

После того, как появится сообщение об успешности сохранения данных, необходимо закрыть это окно и закрыть форму.

Чтобы перейти далее к расчету необходимо нажать на кнопку Далее в окне ввода данных.

При этом программа спросит сохранены ли данные. Если данные были сохранены, то переходим непосредственно к запуску расчета (Рис.8).
